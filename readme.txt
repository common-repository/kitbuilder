Fast and best front-end page builder for WordPress. Make a WordPress site in a minute.

== Description ==
Building a WordPress website for non-developer is a painful job. That's why you need a fast and powerful page builder. KitBuilder will do this job. It's fast and hassles free WordPress page builder with live editing.

From now you don't need to work at back-end just imagine what will look like in front-end. With KitBuilder, you will be able to see your work live. You don't need to imagine any more. 

Our library feature will save your more time. You don't need to double work. You can save a section or page and reuse it later.

== Installation ==

= Minimum Requirements =

* WordPress 4.5 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 5.6 or greater
* MySQL version 5.6 or greater
* WP Memory Limit of 64 MB or greater (128 MB or higher is preferred)


= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents of the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Add New
4. Press the 'Edit with Kitbuilder' button.
5. Enjoy!
